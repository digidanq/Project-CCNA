# CCNA Project / Projekt CCNA / Проєкт CCNA

---

## EN

### Description
This project presents practical networking implementations based on CCNA concepts. It includes designed network topologies and configured devices using Cisco Packet Tracer.

### Features
- Network topology design  
- Router and switch configuration  
- IP addressing and subnetting  
- Static routing  
- Basic network troubleshooting  

### Technologies
- Cisco Packet Tracer  
- TCP/IP  

### How to run
1. Open the `.pkt` files in Cisco Packet Tracer  
2. Analyze the topology  
3. Test connectivity using simulation mode  

### Notes
This project is educational and was created as part of networking coursework.
